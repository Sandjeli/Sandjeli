Quiz: Ecrire une page html5

Niveau 3


1/Quelle balise est utilisée pour créer des liens hypertextes dans HTML ? A. <link>
B. <a>
C. <href>


 2/Quel attribut accompagne toujours la balise <a> pour définir la destination du lien ? A. <src>
B. <alt>
C. <href>


 3/Quelle est la différence entre un lien absolu et un lien relatif ? A. La syntaxe utilisée pour ajouter le texte alternatif
B. Les liens absolus contiennent l’adresse complète, alors que les liens relatifs sont basés sur la structure du site
C. Les liens relatifs nécessitent une extension spéciale


 4/Quel élément dans l'exemple suivant est un lien absolu ? A. <a href="https://example.com">Lien absolu</a>
B. <a href="/page2.html">Lien relatif</a>
C. Aucun des deux


 5/Comment créer un lien relatif vers un fichier appelé page.html se trouvant dans le même dossier ? A. <a href="page.html">Lien relatif</a>
B. <a href="/page.html">Lien absolu</a>
C. <a href="https://page.html">Page</a>



6/Quel code permet d'accéder à un fichier image.jpg se trouvant dans un sous-dossier appelé "images" ? A. <a href="./image.jpg">Image</a>
B. <a href="images/image.jpg">Image</a>
C. <a href="#image.jpg">Image</a>



7/Comment accéder à un fichier dans un dossier parent depuis un sous-dossier ? A. <a href="parent/fichier.html">Fichier parent</a>
B. <a href="../fichier.html">Fichier parent</a>
C. <a href="/fichier.html">Fichier parent</a>.



8/Quelle est la fonction d'une ancre ? A. Charger une page externe.
B. Pointer vers un élément précis sur une page.
C. Ajouter une infobulle au texte.



9/Comment ajouter un identifiant unique à un élément HTML ? A. Avec l’attribut class.
B. Avec l’attribut id.
C. Avec l’attribut src.



10/Comment écrire un lien vers un identifiant appelé section1 ? A. <a href="#section1">Section</a>.
B. <a id="section1">Section</a>.
C. <a href="#top">Section</a>.



11/Quel est l’identifiant spécial pour remonter en haut de la page ? A. #haut
B. #home
C. #top


 12/À quoi sert l'attribut title="" dans un lien ? A. Ajouter un titre à la page
B. Afficher une infobulle au survol du lien
C. Appliquer un style CSS spécifique


 13/Quelle syntaxe déclenche l’ouverture d’une application de messagerie ? A. <a href="message:sana@example.com">Envoyer</a>
B. <a href="mailto:sana@example.com">Envoyer</a>
C. <a href="mail:sana@example.com">Envoyer</a>


 14/Comment insérer un lien qui compose un numéro de téléphone ? A. <a href="phone:+1234567890">Appeler</a>
B. <a href="tel:+1234567890">Appeler</a>
C. <a href="contact:+1234567890">Appeler</a>


 15/Quel attribut force le téléchargement d’un fichier au clic ? A. download
B. href.
C. src.



16/Comment ouvrir un lien dans un nouvel onglet ? A. Avec l’attribut blank="_target"
B. Avec l’attribut target="_new"
C. Avec l’attribut target="_blank"



17/Quelle balise HTML insère une image ? A. <img>.
B. <src>.
C. <image>.



18/Quels sont les attributs nécessaires pour la balise <img> ? A. href, alt.
B. src, alt.
C. src, title.



19/Quelles précautions faut-il prendre avec les noms de fichiers d’image ? A. Utiliser des caractères spéciaux.
B. Remplacer les majuscules et espaces par des tirets, éviter les accents.
C. Toujours écrire en majuscules.



20/Comment afficher une infobulle sur une image ? A. En ajoutant l’attribut tooltip
B. En ajoutant l’attribut src
C. En ajoutant l’attribut title.
