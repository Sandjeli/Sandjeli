Quiz: Ecrire une page html5

Niveau 5


1/Quelle balise permet d'englober tout un formulaire ? A. <fieldset>
B. <div>
C. <form>


2/Quelle est la méthode HTTP par défaut utilisée par la balise <form> ? A. GET
B. POST
C. DELETE


3/À quoi sert l'attribut action dans une balise <form> ? A. Indiquer l'URL de la page de traitement
B. Ajouter un style CSS
C. Définir la méthode HTTP


4/Quelle est la valeur par défaut de l'attribut type dans une balise <input/> ? A. text
B. password
C. email


5/Quel attribut permet de limiter le nombre maximal de caractères dans une balise <input type="text"/> ? A. maxlength
B. limit
C. size


6/Quelle valeur de l'attribut type est utilisée pour un champ permettant de saisir une date ? A. time
B. calendar
C. date


7/Que permet l'attribut readonly dans une balise <input/> ? A. Rendre le champ non modifiable
B. Ajouter une bordure au champ
C. Obliger l'utilisateur à remplir le champ


8/Quelle balise est utilisée pour un bouton qui soumet un formulaire ? A. <input type="submit"/>
B. <button type="button">
C. <input type="button"/>


9/Quelle valeur de type permet de créer un bouton radio ? A. checkbox
B. radio
C. toggle


10/Quelle valeur de l'attribut type permet de créer une case à cocher ? A. tick
B. checkbox
C. radio


11/Quel attribut est utilisé pour fournir une description d'aide dans un champ <input/> ? A. title
B. placeholder
C. alt


12/Quelle valeur de l'attribut type est utilisée pour un champ permettant de saisir une adresse email ? A. url
B. text
C. email


13/L'attribut accept est utilisé avec quel type de champ ? A. <input type="email"/>
B. <input type="file"/>
C. <input type="hidden"/>


14/Quelle valeur de l'attribut type permet de choisir une couleur ? A. color
B. rgb
C. picker


15/Que permet l'attribut checked dans les champs de type checkbox ou radio ? A. Rendre le champ obligatoire
B. Définir la case ou le bouton comme pré-sélectionné
C. Empêcher toute modification


16/Que permet l'attribut min dans un champ <input type="number"/> ? A. Déterminer la valeur minimale possible
B. Ajouter une bordure au champ
C. Valider automatiquement le formulaire


17/Quelle balise est utilisée pour regrouper des champs de formulaire ? A. <group>
B. <fieldset>
C. <form>


18/L'attribut required dans un champ <input/> permet : A. Rendre le champ obligatoire
B. Masquer le champ
C. Désactiver le champ


19/Quelle valeur de l'attribut type permet d'envoyer un fichier au serveur ? A. file
B. upload
C. document


20/Quelle balise est utilisée pour afficher un texte avant un champ de formulaire ? A. <legend>
B. <text>
C. <label>
