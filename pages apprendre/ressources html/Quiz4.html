Quiz: Ecrire une page html5

Niveau 2


1/Que permet de faire la balise <!-- --> ? A. Ajouter du texte visible sur la page.
B. Ajouter des commentaires dans le code HTML.
C. Fermer une balise auto-fermante.


 2/Pourquoi est-il déconseillé d'inclure plusieurs tirets dans un commentaire HTML ? A. Cela rend le fichier trop lourd.
B. Cela peut provoquer des erreurs d’interprétation par le navigateur.
C. Les commentaires deviennent visibles dans le code source.


 3/Quelle combinaison de touches permet d'afficher le code source d’une page dans la plupart des navigateurs ? A. CTRL + U
B. CTRL + S
C. ALT + F4


 4/Quelle est la fonction principale de l’inspecteur d’élément ? A. Créer une nouvelle page web.
B. Modifier le fichier HTML à distance.
C. Examiner et modifier le HTML et le CSS temporairement.


 5/Pourquoi faut-il être vigilant à ce que contiennent vos commentaires dans le code HTML ? A. Parce que les commentaires peuvent être exécutés par le navigateur.
B. Parce qu’ils sont visibles dans le code source par tout utilisateur.
C. Parce qu’ils modifient la mise en page de la page web.



6/Que permet de vérifier l'outil W3C Validator ? A. La vitesse d'une page web.
B. La validité d’une page HTML vis-à-vis du doctype précisé.
C. La compatibilité entre HTML et CSS.



7/Dans quel cas l'indentation est-elle obligatoire en HTML ? A. Elle n’est pas obligatoire mais recommandée pour améliorer la lisibilité.
B. Elle est obligatoire pour les navigateurs modernes.
C. Elle est obligatoire pour activer le CSS.



8/Quel outil permet de tester si votre page web respecte les standards HTML5 ? A. Inspecteur d’élément
B. W3C Validator.
C. F12.



9/Que se passe-t-il si vous modifiez le HTML d'une page dans l’inspecteur d’élément ? A. Les modifications sont enregistrées sur le serveur.
B. Les modifications sont visibles mais perdues au rafraîchissement de la page.
C. La page génère automatiquement une sauvegarde.



10/Pourquoi l’indentation est-elle particulièrement importante en HTML ? A. Elle améliore la lisibilité et facilite la maintenance.
B. Elle optimise la rapidité d’exécution du fichier HTML.
C. Elle s’assure que tous les navigateurs interprètent le fichier correctement.
